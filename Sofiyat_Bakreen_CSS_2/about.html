Nature Photography at its Finest

This gallery is a curated collection of visually calming nature landscapes sourced from free photography libraries. The purpose of the site is to demonstrate CSS layout skills including positioning, floating, display modes, typography, color styling, and web fonts.

The images vary from mountains and forests to beaches and sky formations. Each piece has been arranged using inline-block layouts, with additional CSS techniques such as relative positioning, floating, z-index layering, and sticky navigation.

The website uses a custom Google Font for consistency and aesthetic expression. Colors are applied using hexadecimal, RGB, HSL, and rgba() transparency values to highlight the concepts learned in CSS fundamentals.
